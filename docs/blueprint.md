# **App Name**: KisanConnect

## Core Features:

- Farmer Onboarding: Streamlined phone number OTP login via Firebase Auth and farm profile creation including location (GPS or manual pin) and optional Aadhaar verification for enhanced trust. Profile displays key stats like rating and call count.
- Product Listing: Enables farmers to list produce with multiple input methods including photo upload with AI-powered category suggestions via Google Cloud Vision API, voice input in regional languages transcribed using Google Cloud Speech-to-Text, and SMS listing for basic phones. Listings feature prominent pricing and badges.
- Real-time Consumer Discovery: Allows consumers to browse nearby farms and products, with map and list views, dynamic filtering by category, distance, and special tags like 'Organic'. Features integrated calling.
- Direct Calling: Connect consumers directly with farmers for inquiries and negotiation. Utilizes device dialer, tracks call data.
- Farmer Analytics: Dashboard provides farmers with key statistics on product views, calls received, and listing performance, with visualizations to optimize their sales strategy.
- Community Hub: Offers a dedicated space for farmers to engage with peers, access tips and government updates.
- Admin Moderation: Web dashboard for platform oversight including user management, content moderation, and support ticket handling. Dashboard enables oversight, revenue, usage patterns, and moderation.

## Style Guidelines:

- Primary color: Fresh green (#50D890) evokes a sense of nature and organic produce.
- Background color: Light grey (#F2F4F6) provides a clean, neutral backdrop.
- Accent color: Earthy yellow (#E4A33B) highlights key actions and information.
- Headline font: 'Space Grotesk' sans-serif for headlines, conveys a modern and efficient feel.
- Body font: 'Inter' sans-serif for body text, it provides a clean and readable experience.
- Use clean, intuitive icons from MaterialCommunityIcons to represent categories, actions, and filters.
- Employ smooth, subtle transitions (200-300ms) to create a polished, responsive user experience, along with shimmer effect on loading cards.
